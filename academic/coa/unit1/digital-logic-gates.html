Digital Logic Gates

⏱ 20 min read 📝 Quiz inside 📖 Mano Ch. 2

Binary Logic introduced AND, OR, and NOT as abstract operations on true/false values. This page turns them into physical circuits — the standard gate symbols you'll be drawing for the rest of this course — and adds the operations that make real circuit design possible: NAND, NOR, XOR, XNOR, and the one algebraic rule that ties them all together, DeMorgan's theorem.

The Seven Basic Gates

A logic gate is the physical circuit realization of a Boolean operator. Beyond the three basic operations, four more gates round out the standard set:

  • AND, OR, NOT — the three you already know, now with formal circuit symbols.
  • NAND (NOT-AND) and NOR (NOT-OR) — an AND or OR followed by an inversion.
  • XOR (exclusive-OR) — 1 when the inputs differ.
  • XNOR (exclusive-NOR) — 1 when the inputs match.
Fig 1: Symbol, expression, and truth table for each of the seven basic gates

Every gate symbol follows the same visual grammar: the small bubble on a gate's output means "inverted" — that's the entire visual difference between AND and NAND, and between OR and NOR.

DeMorgan's Theorem

DeMorgan's theorem is the single most useful algebraic identity in digital design:

(A + B)′ = A′ · B′     (A · B)′ = A′ + B′

In words: complementing an expression flips the operator (+ becomes ·, and vice versa) and complements every individual literal — not just the outer expression.

Check for A=0, B=1:
  (A+B)' = (0+1)' = 1' = 0
  A'·B'  = 1·0    = 0
  Match! (True for all 4 input combinations, which is what makes it a theorem.)

NAND and NOR: Universal Gates

NAND and NOR are called universal gates because either one, by itself, can build every other gate — NOT, AND, and OR included. This is a major reason real chips are built almost entirely from NAND (or NOR) gates: manufacturing one gate type repeatedly is far cheaper than manufacturing several different types.

Fig 2: NOT, AND, and OR, each built using only NAND gates
  • NOT from NAND: tie both inputs of a NAND gate together — NAND(A,A) = (A·A)′ = A′.
  • AND from NAND: a NAND gate, followed by a second NAND wired as a NOT.
  • OR from NAND: DeMorgan's theorem in physical form — invert both inputs first, then NAND them: (A′·B′)′ = A+B.

Canonical Form — a Preview

A minterm is a product (AND) term that includes every variable of a function exactly once — uncomplemented if that variable is 1 in a given truth-table row, complemented if it's 0. Writing a function as the OR of all the minterms where it equals 1 is called the canonical sum-of-products (SOP) form.

F(A,B,C) is 1 for rows 1, 3, 5, 6 → F = Σm(1,3,5,6)

F = A'B'C + A'BC + AB'C + ABC'

This form is guaranteed correct but rarely minimal — it's the direct starting point for the graphical simplification method on the next page, Karnaugh Maps.

Common Pitfalls

  • DeMorgan's flips the operator AND complements every literal — complementing only the outer expression is the most common mistake.
  • The bubble is the only visual difference between a gate and its complemented version — AND vs. NAND, OR vs. NOR look identical except for that one bubble.
  • "Universal" means one gate type can build everything — it doesn't mean NAND and NOR are interchangeable with each other without redesigning the circuit.
  • Minterm numbering follows the truth-table row's binary value directly — row 5 (A=1,B=0,C=1) is minterm m5, not an arbitrary label.

Applications

  • Every IC datasheet you'll ever open describes circuits using these seven gate symbols — the actual visual language of digital design.
  • NAND-based design dominates real chip fabrication because of the universal-gate property.
  • XOR gates are the core building block of binary adders and simple parity generators.
